There are several pre-defined shapes that you can use.

| Shape         | Parameters       |
| :------------ | :--------------- |
| Circle        | Radius           |
| Square        | Half-side length |
| Line          | Start, End       |
| Regular N-gon | Sides, Radius    |
| Star          | Tips, Radius     |
| Dot           |                  |
